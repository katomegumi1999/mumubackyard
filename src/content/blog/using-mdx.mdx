这个主题已经预装了 [@astrojs/mdx](https://docs.astro.build/zh-cn/guides/integrations-guide/mdx/) 集成，并在你的 `astro.config.mjs` 配置文件中进行了配置。如果你不想使用 MDX，你可以从配置文件中移除这个集成来禁用它。

## 为什么选择 MDX？

MDX 是一种特殊的 Markdown 变体，它支持嵌入 JavaScript 和 JSX 语法。这使得你可以将 [JavaScript 和 UI 组件混合到你的 Markdown 内容中](https://docs.astro.build/zh-cn/guides/markdown-content/#mdx-features)，用于创建交互式图表或提示框等。

如果你已经有使用 MDX 编写的内容，这个集成将使你轻松地迁移到 Astro。

下面是如何在 MDX 中导入和使用 UI 组件的示例。
当你在浏览器中打开此页面时，你应该看到下面的可点击按钮。

import HeaderLink from '../../components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('点击了！')">

</HeaderLink>
